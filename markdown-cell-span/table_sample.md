# Markdown テーブルセル結合のサンプル

## 横（行）の結合

```md
|  A            |  B        |  C            |
|  ------ ----- |  ---- --- |  ------ ----- |
|  通常          |  通常      |  通常         |
|  右結合        |           |               |
|               |           |  左結合        |
```

| A      | B    | C      |
| ------ | ---- | ------ |
| 通常   | 通常 | 通常   |
| 右結合 |      | 通常   |
| 通常   |      | 左結合 |

## 縦（列）の結合

```md
| A                    | B                    | C    |
| -------------------- | -------------------- | ---- |
<td rowspan="2"> 結合   | 通常                 | 通常  |
|                      <td rowspan="2"> 結合   | 通常 |
| 通常                 |                       | 通常 |
```

| A                    | B                    | C    |
| -------------------- | -------------------- | ---- |
<td rowspan="2">結合 | 通常                 | 通常 |
|                      <td rowspan="2">結合 | 通常 |
| 通常                 |                      | 通常 |
